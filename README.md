# mginx

### make server like nginx

## Makefile commands

---

### RUN COMMANDS

### make run

run webserv

### make test

run webserv && run client

### make tester

run webserv && open new terminal window(for running 42tester)

---

### BASICS COMMANDS

### make

compile server, client

### make re

fclean && make again

### make clean

remove object files

### make fclean

remove object files, log file and execute file
